To install the firmware, perform the following steps:

*     Click on the Start button and select Run from the menu.
*     In the Open edit box, type a:\fwload.exe and press the OK button.
*     If the IP address shown does not match the IP address of the 
      tracker to receive the new firmware, change the shown address.
*     Click on "Install Firmware" and respond to any questions.
*     When the firmware is properly installed, reboot the tracker.
